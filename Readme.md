# Калькулятор (CalcMini)

Простой консольный калькулятор для арифметических операций.


## Требования
- Python 3.8+


## Установка
```bash
git clone https://github.com/PS-Bot-Maker/CalcMini.git
cd calc-mini
```

## Запуск
```bash
python calc_mini.py
```

## Примеры
```
Введите первое число: 10
Введите оператор: *
Введите второе число: 5
Результат: 50.0
```

## Поддерживаемые операции
- `+` — сложение
- `-` — вычитание
- `*` — умножение
- `/` — деление


## Ошибки
- «Ошибка! Деление на ноль!»
- «Ошибка! Неизвестная операция!»
- «Ошибка! Введите число.»


## Структура
- `calc_mini.py` — основной код
- `validations.py` — валидация ввода
- `config.py` — настройки и операции
- `calc_mini_pytest.py` — тесты
- `README.md` — документация
